# Assets Directory

This directory should contain the following image files for the bot to function properly:

## Required Fortune Images:
- `大吉.png` - Great fortune
- `小吉.png` - Small fortune
- `末吉.png` - Uncertain fortune
- `平.png` - Average fortune
- `小凶.png` - Small misfortune
- `大凶.png` - Great misfortune

## Required Other Images:
- `mao.jpg` - Default image for private message responses

## How to Add Images

1. Create or find appropriate images for each fortune type
2. Save them with the exact names listed above in this directory
3. Recommended image size: 512x512 or similar square dimensions
4. Supported formats: PNG for fortune images, JPG for mao image

**Note:** The bot will not work properly without these images. Please add them before running the bot.
